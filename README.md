1. Python Quiz Game (Questioning.py)
An interactive, high-stakes trivia game where players answer questions to build up a virtual "prize pool."
 * Key Features:
   * Multiple-choice questions with real-time scoring.
   * Case-insensitive input handling (accepts "YES" or "yes").
   * Dynamic prize calculation based on correct answers.
 * Concepts Used:
   * print() and input() for user interaction.
   * if/elif/else statements for game logic.
   * for loops to display answer options from a list.
   * Variable manipulation to track the total prize money.
