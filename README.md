# Beginner Python Projects Collection 
Welcome to my repository of Python practice projects! This collection showcases my journey in learning Python through interactive games and utility scripts.
## Projects Included
1. Python Quiz Game (Questioning.py)
An interactive, high-stakes trivia game where players answer questions to build up a virtual "prize pool."
 * Key Features:
   * Multiple-choice questions with real-time scoring.
   * Case-insensitive input handling (accepts "YES" or "yes").
   * Dynamic prize calculation based on correct answers.
 * Concepts Used:
   * print() and input() for user interaction.
   * if/elif/else statements for game logic.
   * for loops to display answer options from a list.
   * Variable manipulation to track the total prize money.
# How to Run the Projects
 * Prerequisites: Ensure you have Python installed on your system.
 * Clone the Repo:
   git clone https://github.com/Aroos-jabeen/Beginner-Python-Project.git
 * Navigate to the folder:
   cd Beginner-Python-Project
 * Run the Game:
   python Questioning.py
